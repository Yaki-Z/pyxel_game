- 使い方

  - 起動方法は`python app.py`
  - 5 行目の`WINDOW_WIDTH`と 6 行目の`WINDOW_HEIGHT`を変えて画面の縦横を設定
  - 8 行目の`NUM_CIRCLE`で丸の数を設定
  - 9 行目の`NUM_TRIANGLE`で三角の数を設定
  - 11~13 行目で各種色を設定
